# ポートフォリオサイト 技術スタック

## コア

| カテゴリ | 技術 | 備考 |
|---|---|---|
| フレームワーク | Astro | 静的サイト生成。View Transitions 標準対応 |
| 言語 | TypeScript | 型安全 |
| パッケージマネージャー | pnpm | 高速・ディスク効率 |

## コンテンツ管理

| 対象 | 方式 | 備考 |
|---|---|---|
| Works | Markdown | Astro Content Collections で管理 |
| Blog | Markdown | Astro Content Collections で管理。Zenn連携しやすい |

- microCMS は不使用（完全ローカル管理に移行）

## スタイリング

| カテゴリ | 技術 | 備考 |
|---|---|---|
| コンポーネントスタイル | Astro scoped `<style>` | JSゼロ。コンポーネントと同居 |
| グローバルスタイル | CSS変数（Custom Properties） | design-tokens をCSS変数として定義 |
| フォント | Google Fonts | 具体的な書体はデザインに合わせて選定 |

## アニメーション

| 種類 | 実装方法 |
|---|---|
| スクロールフェードイン | Intersection Observer + CSS animation |
| ホバーエフェクト | CSS transition |
| ページ遷移 | View Transitions API（Astro標準） |
| Hero背景 | CSS animation（浮遊演出） |

- ライブラリ不使用。CSS + 軽量JSのみで実装

## UI

| カテゴリ | 技術 | 備考 |
|---|---|---|
| アイコン | astro-icon + Iconify | SVGインライン化。JSゼロで軽量 |
| ダークモード | Light / Dark 両対応 | CSS変数の切り替えで実装。design-tokens 定義済み |

## 開発ツール

| カテゴリ | 技術 | 備考 |
|---|---|---|
| Linter | ESLint | Astro用プラグインあり |
| Formatter | Prettier | Astro用プラグインあり |

## インフラ・デプロイ

| カテゴリ | 技術 | 備考 |
|---|---|---|
| ホスティング | Vercel | 自動デプロイ・プレビューデプロイ対応 |

## 追加機能

| 機能 | 技術 | 備考 |
|---|---|---|
| OG画像自動生成 | satori | ブログ記事ごとにOG画像を動的生成 |
| サイトマップ | @astrojs/sitemap | ビルド時に自動生成。SEO基本対策 |

## 主要パッケージ（想定）

```
astro
typescript
@astrojs/check
astro-icon
@iconify-json/*        # 必要なアイコンセット
eslint
eslint-plugin-astro
prettier
prettier-plugin-astro
@astrojs/sitemap
satori
sharp                  # OG画像の画像処理用
```

## アーキテクチャ概要

```
src/
├── components/        # Astroコンポーネント
├── layouts/           # ページレイアウト
├── pages/             # ルーティング
│   ├── index.astro    # トップページ（Hero/Profile/Works/Blog/Footer）
│   ├── works/[slug].astro
│   ├── blog/[slug].astro
│   └── blog/index.astro
├── content/           # Content Collections
│   ├── works/         # Works記事（.md）
│   └── blog/          # Blog記事（.md）
├── styles/            # グローバルCSS・design tokens
└── assets/            # 画像等の静的アセット
```
